* [RxJS 6.4.0](https://www.npmjs.com/package/rxjs/v/6.4.0)
* [Typescript 3.5.3](https://www.typescriptlang.org/index.html#download-links)
* [Webpack 4.39.2](https://www.npmjs.com/package/webpack/v/4.39.2)
